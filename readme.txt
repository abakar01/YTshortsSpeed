simple speed controller for youtube shorts

This Chrome extension allows you to control the playback speed of YouTube Shorts with a custom menu featuring various speed options (e.g., 0.1x to 3x).

How to Install:
Download the extension files from this repository.

Open Google Chrome and go to chrome://extensions/.

Enable Developer mode in the top-right corner.

Click on Load unpacked and select the folder with the extension files.

Enjoy customizing your playback speed on YouTube Shorts!
